# Habit Tracker

A simple habit tracking web application that helps users build and maintain daily habits.

## Features
- Add and manage daily habits
- Track habit completion
- Goals Tracker
- Simple and clean user interface
- Fast development setup with React and Vite

## Tech Stack
- Frontend: React
- Build Tool: Vite
- Language: JavaScript
- Styling: CSS

## Installation

1. Clone the repository
git clone https://github.com/your-username/habit-tracker.git

2. Navigate to the project folder
cd habit-tracker

3. Install dependencies
npm install

4. Run the development server
npm run dev
